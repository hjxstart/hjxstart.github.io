我的笔记 notes
================

## 安装
```bash
git clone git@github.com:hjxstart/notes.git
```

## hexo

1.安装自定义主题
```bash
git clone https://github.com/hjxstart/3-hexo.git blog/themes/3-hexo
```

2.安装 hexo-cli
``` bash
npm install -g hexo-cli
```

3.创建一篇新文章
``` bash
cd /blog
hexo new "My New Post"
```

4.初始化
``` bash
npm run 
```
