    发布时间 : 2021-03-14 08:31
    阅读 :
    1. 1.node环境搭建
    2. 2.安装 hexo-cli
    3. 3.文件夹下初始化博客
    4. 4.创建一篇新文章

    6. 6.生成博客文件
    7. 7.安装 git 自动化部署
    8. 8.修改 _config.yml 文件
    9. 9.推送到远端服务器
    10. 10.下载 yilia 主题
    11. 11.修改 _config.yml 文件

    1.node环境搭建

    $ node -v

    2.安装 hexo-cli

    $ cnpm install -g hexo-cli

    3.文件夹下初始化博客

    $ sudo hexo init

    4.创建一篇新文章

    $ hexo new "My New Post"

    5.本地运行预览

    $ hexo server

    6.生成博客文件

    $ hexo generate

    7.安装 git 自动化部署

    $ npm install hexo deployer-git --save

    8.修改 _config.yml 文件

    deploy:
  type: git
  repo: https://github.com/xxx/xxx.github.io.git
  branch: master

    9.推送到远端服务器

    $ hexo deploy

    10.下载 yilia 主题

    $ git clone https://github.com/litten/hexo-theme-yilia themes/yilia

    11.修改 _config.yml 文件

    themes：yilia
