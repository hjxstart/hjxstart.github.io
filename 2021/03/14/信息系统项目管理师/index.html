信息系统项目管理师

第1章 信息化和信息系统

1.1 信息系统和信息化

  1. 信息：信息是能够用来消除不确定行的东西；
  2. 信息技术：是为了解决信息的处理问题而发展起来的技术；
  3. 信息系统：是输入数据，通过加工处理，产生信息的系统；
  4. 信息化：宏观概念。通常是指现代信息技术应用，特别是促成应用对象或领域发生转变的过程；是推动经济社会发展转型的历史过程。

1.1.1 信息

信息特性：
宏观性、普遍性、无限性、动态性、相对性、依附性、变换性、传递性、层次性、系统性、转化性。

信息质量属性

  1. 精确性，对事物状态描述的精准程度。
  2. 完整性，对事物状态描述的全面程度，完整信息应包含所有重要事实。
  3. 可靠性，指信息的来源、采集方法、传输过程是可以信任的，符合预期。
  4. 经济性，指获得信息的时刻与事件发生时刻的间隔长短。
  5. 可验证性，指信息的主要性质属性可以被证实或者证伪的程度。
  6. 安全性，指在信息的生命周期中，信息是否可以被非授权访问。

1.1.2 信息技术

信息只有流动起来才能体现其价值，因此信息的传输技术（通常指通信，网络等）是信息技术的核心。

图

1.1.3 信息系统

信息系统可以是手工的，也可以是计算机化的，本书中讨论的系统是计算机化的信息系统。信息系统的组成部件包括硬件、软件、数据库、网络、存储设备、感知设备、人员以及把数据处理成信息的规程等。

  1. 开放性，系统的开放性是指系统的可访问性。
  2. 脆弱性，这个特性与系统的稳定性相对应，即系统可能存在着丧失结构、功能、秩序的特性。
  3. 健壮性，系统具有的能够抵御出现非预期状态的特性，也称鲁棒性。

信息系统生命周期

  • 系统规划阶段（可行性研究）
  • 系统分析阶段（需求分析）
  • 系统设计阶段（概要设计，详细设计）
  • 系统实施阶段（编码，测试）
  • 系统运维阶段

系统分析、系统设计、系统实施属于开发阶段

1.1.4 信息化

  1. 通常是指现代信息技术应用，特别是促成应用对象或领域发生转变的过程；是推动经济社会发展转型的历史过程。
  2. 核心是通过全体社会成员（主体）努力，在经济和社会各领域充分应用基于现代信息技术的生产工具（手段），创建信息时代社会生产力，并推动生产关系和上层建筑改革（途径），使国家的综合实力，社会文明素质和人民生活质量全面提升（目标）。
  3. 产品信息化、企业信息化、产业信息化、国民经济信息化、社会生活信息化。
  4. 国家信息化体现6要素：人才、法规、资源、网络、技术应用、产业

1.2信息系统开发方法

信息系统的开发需要大量的人力、物力、财力和时间的投入。在系统开发时为了更好低控制时间、质量、成本，并使用户满意，除了技术、管理等因素外，系统开发方法也起着很重要作用。

  1. 结构化方法。
  2. 面向对象方法。
  3. 原型化方法。
  4. 面向服务的方法。

1.2.1结构化方法

结构化分析、结构化设计、结构化程序设计

图

1.2.2 面向对象方法（Object-Oriented, OO）

  1. 客观的世界每个对象都有自己的运动规律和内部状态，都属于一个对象类，是对象类的一个元素。复杂对象可由相对简单的各种对象组以某种方式组合而成。
  2. 分析、设计、实施三个阶段没有明显缝隙

1.2.3 原型方法

  1. 可以使系统开发的周期缩短，成本和风险降低，速度加快，获得更高的综合开发效益。
  2. 以用户为中性开发系统，用户参与程度大大提高，开发的系统符合用户需求，增加客户的满意度，提高成功率。
  3. 由于用户参与了系统开发的全过程，对系统的功能和结构容易理解和接受，有利于系统的移交和运维。

    图

1.2.4 面向服务的方法（Service-Oriented，SO）

OO的应用构建在类和对象之上，随后发展起来的建模技术将相关对象按照业务功能进行分组，就形成了构件的概念。对于跨构件的功能调用，则采用核心的形式暴露出来。

  1. 进一步接口的定义与实现进行耦合
  2. 提高系统的可复用性，信息资源共享和信息系统之间的互操作性。

1.3 常规信息系统集成技术

系统集成是指将计算机软件、硬件、网络、通信等技术和产品集成为能够满足用户特定需求的信息系统，包括总体策划，设计开发，实施，服务及保障。

  1. 网络标准与协议
  2. 网络存储技术
  3. 网络设备
  4. 网络接入技术
  5. 网络服务器
  6. 数据库管理系统

    1.3.1 OSI模型

    图

1.3.2 网络接口层协议（IEEE802）

  1. IEEE802.11(无线网局域网WLAN标准协议)
  2. IEEE 802.3 标准以太网 10MB/S 同轴线缆
  3. IEEE 802.3u 快速以太网 100MB/S 双绞线
  4. IEEE 802.3u 千兆以太网 1000MB/S 光纤或双绞线

1.3.3 网络层协议

  1. IP:提供端到端的分组分发协议。
  2. ICMP:网际控制报文协议。
  3. IGMP:网际组管理协议。
  4. ARP:地址解析协议，动态完成IP地址的转换，RARP。

1.3.4 传输层协议

  1. TCP(面向连接，可靠协议，缺点：传输效率不高)
  2. UDP（）

1.3.5 应用层协议

  1. FTP:文件传输协议
  2. TFTP:快速网络传输协议，建立在UDP协议上
  3. HTTP:超文本传输协议
  4. SMTP:简单邮件传输协议
  5. DHCP:主机动态配置协议，建立在UDP协议之上
  6. Telnet:远程登录协议
  7. SNMP:简单网络管理协议

1.3.6 网络设备

  1. 中继器：对接受信号进行再生和转发，扩展传输距离，属于物理层。
  2. 集线器：多端口中继器(HUB)，属于物理层。
  3. 网桥：根据帧物理地址进行网络之间的信息转发，网络链路层。
  4. 路由：通过逻辑地址进行网络信息转发，网络层。
  5. 交换机：网桥+路由，2-7层。
  6. 网关：局域网连接，4-7层。

1.3.7 网络服务器

网络服务器是指在网络环境不运行相应的应用软件，为网上用户提供共享信息资源和各种服务的一种高性能计算机，或者计算机集群，逻逻辑上仍是一台计算机。

1.3.8 网络存储技术

  1. DAS(直接附加存储)：直接连接在服务器上(I/O总线)，不带任何操作系统。
  2. NAS(网络附加存储)：可以在网络中任意建立存储，支持即插即用。
  3. SAN(存储区域网络)：通过专用交换机将磁盘阵列与服务器连接的高速子网。FC SAN、IP SAN、IB SAN

1.3.9 网络接入技术

  1. 有线接入(PSIN ISDN ADSL FTTX+LAN HFC)
  2. 无线接入技术(WI-FI、3G、4G、5G)
  3. 4G TD-LET(100mb)、FDD LTE(150mb)、5G(1GB)

1.3.10 数据库技术

Oracle、MySql、SqlServer、MongoDB、Hbase等
