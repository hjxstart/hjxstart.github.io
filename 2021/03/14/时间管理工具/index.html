时间管理工具

04 养成坏习惯

不怕坏习惯，怕坏习惯失控。

  1. 使用场景：不容易停下来；不能切底戒掉。
  2. 具体方法：为坏习惯安排出特定时间；把坏习惯当作完成任务的奖励。
  3. 控制剂量：每天或每周安排出时间；监控自己是否过量。
  4. 无聊是根本问题：优先解决无聊问题；用好习惯充实生活。
  5. 注意事项：和平共处；练习停止；像培养好习惯一样对待。

05 倒逼法

人的成长要靠倒逼

  1. 早睡的动力：晚醒会影响精神状态；长期晚睡会影响到身体健康。
  2. 倒逼入睡的方法：让自己没有余地，没有妥协可能性；Wake闹钟的设置。
  3. 问题：通常情况下我们有一个任务。我们手头的事情通常情况下都比这个要长，也许我们有100天的时间，也就是说截止日期是在一百天后。多数人是前90天没有认真作事情，最后十天突击，专心在做事情。
  4. 规则：在目标日期之后，绝对不花时间来做这件事情（目标日期不是截止日期）
  5. 突破自己：远离自己的舒适圈 图

06 具象法

坚持自上而下的改变

  1. 自上而下：观念上的改变。
  2. 自下而上：行为上的变化。
  3. 不同角度：情绪；认知；行为。、
  4. 感受一下：向前一小步，文明一大步。
  5. 分析：时间管理不是励志，不是鸡汤；时间管理是非常具体的。

07 要事优先

只要是你打算作的事情，不管重不重要的事情

  1. 理念：在生活中最重要的是什么？为自己创建自由，具体表现；拥有你自己。
  2. 为自己创造财富：健康；资产；人际。

08 五分钟清醒术

身体醒了大脑就跟着醒了

  1. 问题：起床后出现困倦感。
  2. 两个赖床阶段：躺在床上不想东；下了床想回去。
  3. 解决方法1：手指脚趾动法。局部带动全身的原理。
  4. 解决方法2：肠胃苏醒法。白开水带动肠胃苏醒。
  5. 其他补充方式：刷牙、洗脸、吃东西、活动身体和思考。

09 20/80法则

和时间管理没有半毛钱关系

  1. 没有捷径：如果你真的花了20%的时间创造了80%的价值，他并不代表你一定过好了，并不代表剩下的80%的时间你就可以什么都不做了；思考一下自己最重要的哪些价值应该在什么时间产出；抓大放小，而不是面面俱到。
  2. 一天之际：你一天当中最精华的2到4个小时的时间是在那段？
  3. 关键任务：认清自己最重要的任务有哪些？
  4. 基础习惯：你的日常习惯当中哪些是最基础最重要的？

用量

  1. 时间：每月或每季度一次，通常是自己的时间规律发生变化时才有必要重新思考。
  2. 任务：每日、每周、每月或每年思考，根据实际需要的频次来决定。
  3. 习惯：根据自己的习惯养成情况决定，一项重点习惯养成，再重新调整。
  4. 注意事项：20/80是指导原则；20/80比例仅供参考；必须找到衡量标准。

10 三条命坚持法

坚持，没有名怎么行？

  1. 什么时候应用：每天都要坚持做一件事。
  2. 原因：每天能做的事情有限；无聊做什么很难做到每天做到每天都一样。
  3. 为什么要用三条命坚持法：重点是帮助你坚持。
  4. 设定周目标：七天当中几天做到；推荐四天；MAC：最小有成就感的改变。

11 沐浴自控法

用洗澡来锻炼自己的自控力

  1. 问题：晚间停不下来；原因是陷入来刺激中，刺激一旦开始，我们很难脱离出来。
  2. 解决方法：更早的时间；思路是在我们还未陷入到刺激当中，原理刺激。
  3. 沐浴自控法：洗完澡之后，就可以开始下一个读书学习活动。
  4. 如何操作：睡前1.5小时洗澡。

12 晚间二分法

放松休闲，充电提升，而不是耽误

  1. 晚间时光如何利用：晚间的需求（娱乐休闲，提升充电）；困难和障碍（娱乐休闲停不下来，提升充电学不进去）。
  2. 两种需求都应该被满足：晚上的时间一分为二。
  3. 有安排玩的痛快：为刷手机安排出时间。
  4. 分界线：以洗澡为界，分成两个部分。
  5. 睡前学习提神：读书学习，准备第二天，放松。

13 三只青蛙法

练习自我承诺和兑现承诺

  1. 关键问题：今天你最重要的一件事情是什么？
  2. 青蛙的条件：自我要求而非外部约定；完成之后的轻松感和成就感。
  3. 只有一件：过程，逐渐增加保持的状态。
  4. 练习的内容：自我控制的能力，每天规划一个重点，每天搞定一个重点。

14 随手记

贵人多忘事，只因记心中

  1. 当想法出现在大脑里：否定；忽略；遗忘；压抑。
  2. 灵光一闪：记录下来，在合适的时间进行推进。
  3. 随手记的方法：What：想法；When：立刻；Where：纸或手机；How：最简单的文字。
  4. 时间管理者的标志-随手记：时间管理者能有意志的管理自己；水平越高的人随手记的数量越多。

15 状态优先法

通过做任务来调节自己的状态

  1. 状态时好时坏：状态好的时候，什么都能做到；状态不好的时候，什么都不想做。
  2. 状态好的时候可以选择一些任务困难一点的任务来做。

16 蛋糕切分法

如何一步一步的搞定事情

  1. 危难情绪：焦虑 -> 逃避。
  2. 如何改变：改变自己；改变任务；改变感知。
  3. 通过切分改变感知：仅仅关注下一步行动（有明确的结果，能在30分钟以内完成）。
  4. 核心思想：关注过程，而非结果。

    17 Getting Things Done (GTD)

最系统全面的工作方法

  1. 适合人群：忙碌者严格执行。

18 自然计划法

通过有效思考来做计划，适合短期任务

  1. 自然计划过程：定义结果；头脑风暴；组织整理；心里预演。
  2. 定义结果：核心任务，搞清楚任务做成的想要样子，做到的程序，得到的结果。
  3. 头脑风暴：思考能够实现目标的手段。
  4. 组织步骤：按照先后顺序整理。
  5. 心里预演：畅想整个过程。

19 72小时原则

错过了时机不做也就不做了

  1. 理念出外：书名《小狗钱钱》
  2. 从想法到行动，各种拖延的可能；否定，自己扼杀一个想法；忽视，没有引起足够充实；压抑，选择性遗忘；遗忘，切底遗忘。
  3. 在72小时之内给想法找个归宿：下一步行动；提醒；留存；决定放弃。
  4. 注意：使用随手记工具。

20 帕金森法

比番茄工作法更实用

  1. 帕金森定律：当人只有有限的时间时，效率是比较高的。
  2. 提前预计时间花费：知道任务多长时间，实现执行任务的高效，单位为分钟。
  3. 执行任务准备倒计时时钟：增加你的紧迫感；预测任务的花费时间，在执行时计时。

21 番茄工作法（不推荐）

  1. 如何应用：确定任务–做25分钟–休息25分钟；将任务按照时间推进。
  2. 蛋糕切分方法的变种：处理为难情绪；改变感知；从时间维度切分。
  3. 核心思想：把任务变的小一点，短一点，小一点。

22 小红花法

用打卡激励自己坚持

  1. 方法简单之极：日历记录，当天是否做到这个习惯。
  2. 核心目的：通过对自己的习惯记录，把更多的精力关注和思考这个习惯，促使习惯的萌芽和坚持。

23 结合健身法

最好的健身是没觉得自己在健身

  1. 将运动建立在生活上：远足；园艺；修建围栏。
