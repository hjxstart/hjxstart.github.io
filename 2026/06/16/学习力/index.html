颠覆脑力，让你更聪明的哈佛学习力

1 认知升级篇

# 21世纪的成功的四大特性
不可预测--快
无法复制
百发齐放--优秀的标准不再单一，只要有一技之长，每一个人成功的可能性非常宽广
难以跨越--阶级的难以跨越是表象，而思维层次差距是阶级难以跨越的背后原因
# 学习力
我们过多的纠结于知识内容本身的记忆，过多的关注如何模仿成功人士的行为，过少的去忽略学习背后的思考和思维逻辑；所以21世纪的学习力应该是对于知识，内化、应用、了解自己的特定和特长并且加以发挥的学习力，这才是最关键的。
# AI时代意味着什么？
掌握信息本身不再重要，高效的去归纳、总结，甚至创造知识的能力更为重要。

1.1 解读“聪明”：开挂的背后，真正的聪明人如何学习？

# 聪明
真正的聪明不在于死记硬背，而是对于生活的观察和思考。

# 智慧
真正有智慧的人，不仅仅需要每日持续的精进，更需要对于这个社会时事的把握和了解。（每日一知道；家事国事天下事，事事放心上）

# 总结
学习了不仅仅只有能力，更有动力和毅力；
现阶段，中国人更多地关照学习的技能技巧本身，而忽略了对于动力和毅力的关注。

2 学习动力篇

2.1 成长型思维：撕掉标签，将缺点转变为成长的突破口

## 我是一个什么样的人？
# 僵固型思维模型 (stiff mindset)
1. 我是一个非常聪明的人
2. 我是一个数学特别差的人
# 成长型思维模型 (growth mindset)
1. 我有潜力在数学方面更加精进
2. 我非常努力

## 如何培养和锻炼自己的成长型思维？
# 如何直面自己？
1. 永远不要给自己下定义
2. 永远不要轻信局限自我的评论
# 如何面对生活的挑战
1. 思维模式转换：学习机遇还是人生困境？
2. 热爱所有的挑战与失败；
3. 自我归因还是环境归因。
# 如何直面评论意见？
1. 你接受负面评论的能力有多强？
2. 负面评论--它们的价值比正门评论更大。

## 建议与推荐：
# 每天晚上在睡觉之前抽5分钟的时间，去反思和回顾一下当天，拿笔写一写：
1. 什么时候自己陷入了僵固型思维的怪圈？
2. 什么时候可以可以的改变一下自己的思维模式？

《终身成长》LEARN OR DIE

2.2 激情与使命：找到动力，让长远目标不断支撑你学习

# 空心病--北京大学30%以上的新生有厌学倾向。
看上去很努力，每台很勤奋，但是从来都没有生活的意义、价值感和使命感。

# 激情
可以让自己付出100%热情、激情，哪怕失败了、跌倒了也愿意持之以恒做下去的真正想要做的事情。
# 使命感
建立于激情之上，真正的热爱某件事，能意识到这个社会的痛点，拥有愿意投身这件事的决心和毅力；
把这个世界、社会的问题当作是自己的使命感和责任感；
使命感对于成功不可或缺，它是能够让我们真正走上成功之路的一个必要因素；
真正的持续有效的利己，只能通过利他到达；只有利他才能利己。

## 课后反思
# 仔细反思一下你过去的认识
1. 你真正热爱的事物是什么？你擅长的是什么？
2. 你能够凭借你的一己之力给身边的人带来什么样的帮助？

2.3 满足感：学会激励，在学习中收获长久的满足感

## 自我激励
# 优点
1. 不会随着时间的推移而减弱；
2. 给人带来高幸福感与满足感；
# 缺点
1. 仅仅适用于自己真正喜爱的事物；
2. 对于其他工作较难实施。

## 外接奖励
1. 以物质、精神鼓励等形式对一件事情进行肯定；
2. 外界奖励被普遍运用在现在的学习、教学中。
# 优点：1、简单快捷；2、条件反射。
# 缺点：1、需要持续的外接刺激，奖励消失，行为难以持续保持；2、有可能让我们失去对于事物本身的热情。
# 最好的激励方式=自我激励+外接鼓励

最好的平衡，是基于自我激励、自我价值认同，并且稍微使用外界激励辅助。

## 课后思考
近一年来，你有没有什么特别想要完成的目标？请你就这个目标做一个描述，并制定一下你打算如何利用自我激励和外界激励来达成。
