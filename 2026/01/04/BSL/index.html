## 目录
# dir 查看目录
dir /?
dir # 查看当前目录
dir /s # 递归显示当前目录及子目录
dir /s/p # 翻页
dir /a # 显示隐藏文件
dir Desk*  # 可以用正则来查看目录
# md 创建目录
md test # 创建test目录
md a\b\c\d # 创建递归目录
# rd 删除目录
rd /s test # 删除test目录树，-r差不多
rd /s/q test # 删除test目录，不用问 -rf差不多


## 文件

cd> aa.txt # 创建文件
cd> a\aa.txt # 创建a目录和在a目录下创建aa.txt
# echo 编辑文件内容
echo hello > aa.txt
echo hi >> aa.txt
# type 查看文件内容
type aa.txt
# copy 复制文件 
copy aa.txt test # 将aa.txt拷贝到test目录下
copy aa.txt bb.txt # 将aa.txt拷贝一份并重命名为bb.txt
#  ren 重命名文件
ren aa.txt cc.txt # 将aa.txt重命名为cc.txt
# del 删除文件
del aa.txt # 删除文件

    Window文件系统

    Windows支持的文件系统格式

    # CDFS、UDF、FAT12、FAT16、FAT32、NTFS
CDFS: CDFS是一个支持CD-ROM（只读光盘）文件的只读文件系统，最大支持的文件大小为4GB，最多支持65535个目录
UDF: UDF（通用磁盘格式文件系统）主要提供了对DVD文件的支持
FAT: FAT(File Allocation Table 文件分配表)，诞生与1977年，最初是为软盘设计，主流的文件系统(FAT16/32/64)
	特点：单个文件最大为4G，最大文件数量268/435/437，分区最大容量8TB，可在多种操作系统读写
	缺陷：当文件被删除并且在同一位置写入新数据，数据片段是分散的；浪费磁盘空间；磁盘利用率低；不支持长文件名
NTFS: NTFS(New Technology File System 新技术文件系统)，一种比FAT32功能更强大的文件系统
	从Windows2000之后的Windows系统的默认文件系统都是NTFS，而且这些Windows系统只能够安装在NTFS格式的磁盘上
	特点：
		1、安全性：能够轻松指定用户访问某一个文件或目录、操作的权限大小；
			能用一个随机产生的密钥吧一个文件加密，只有文件的所有者和管理员掌握解密的密钥，其他人即使能够登录到
			系统中，也没有办法读取他。
		2、容错性：使用了一种称为事务登录的技术跟踪对磁盘的修改。因此，NTFS可以在几秒钟内回复数据
		3、稳定性：文件不易收到病毒和系统崩溃的侵袭。
		4、向下的可兼容性：可以存取FAT文件和HPFS文件系统的数据；如果是软盘，它将自动采用FAT文件系统
		5、可靠性：把重要交易作为一个完整交易来处理，只有整个交易完成之后才算完成，这样可以避免数据丢失
			如向NTFS分区中写文件时，会在内存中保留一份拷贝文件，然后检查向磁盘中所写入的文件是否与内存拷贝的
			一致。如果两者不一致，操作系统就把相应的扇区标为坏扇区而不再使用它，然后用内存中保留的文件拷贝
			重新向磁盘上写文件。
		6、大容量：NFS磁盘的单个文件最大为16EB-1KB，即(2^64)-(2^10)。
		7、长文件名：允许长达255个字符的文件名。FAT规定文件名为8个字符扩展名为3个字符

C:\Windows\System32\config\SAM


## 用户
# 查看
whoami /user # 查看user用户的UUID
wmic useraccount get name,sid # 查看所有用户的UUID
net user # 查看所以用户
net user mz # 查看mz账号的详细信息
net user administrator # 查看administrator账号的详细信息
net user administrator /active:yes # 激活账号
# 增删改
net user mzzz 123 /add # 创建用户
net user mzzz /del # 删除用户
net user mzzz "" # 重置密码为空
net user mzzz 123 # 设置用户密码为123


## 组
net localgroup # 查看本地所有的组
net localgroup read # 查看本地组read的成员
net localgroup mzs /add # 创建mzs组
net localgroup mzs /del # 删除mzs组


## 用户&组
net localgroup read mz /add # 将mz用户添加到read组中
net localgroup read mz /del # 将mz用户从read组中删除
