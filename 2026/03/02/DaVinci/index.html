入门篇

视频制作流程概述

剪辑

# 剪辑：把拍摄的素材拼接在一起，编排结构，控制演出节奏。无论是电影项目还是短视频，最重要的东西都是相通的。基本内容大致有如下：
素材管理
创建工程
创建时间线
挑选素材
剪辑
添加效果
添加音频
添加字幕
导出

素材管理

# 首先你需要了解，每一个项目我们都需要将素材分门别类存放在文件夹中。
# 文件夹结构属于基本管理

创建工程、时间线和挑素材

# 首次打开达芬奇，我们需要在项目这个位置，添加一个项目库，然后新建一个项目。
# 打开项目第一步，首先要去到右下角齿轮，去设置一下时间线的帧速率设置成25帧，分配率保持默认的1920x1080，点保存
# 第二步去到媒体窗口的左上角，找到我们素材那个文件夹存放的位置，把视频素材文件夹拖到Master下，不更改项目频率。
# 时间线上挑选素材：去到剪辑窗口，把视频素材全部放到时间线上，可以将不同的素材隔开进行挑选，对素材进行一些插头去尾部的操作保留需要的素材，最后要更精细的去看和挑选一遍素材；
# 媒体池挑选素材，媒体池里可以看每一条素材，然后在里面打入点和出点，打完点之后就可以拖到时间线上。

剪辑

# 剪辑流程是一个偏复制的操作

添加效果

# 变速
D: 无极变速
更改片段速度: 整倍变速

# 放大
选中素材，到右上角的检查器去做变换；也可以到画布的左下角的框，切换成手动调整。

# 特效

添加音频

# 将音频文件夹拖拽到master下，然后切换回剪辑面板
# 添加音乐，选择第一个音乐双击，然后在这个源素材监视器里，对它打入出点，框选一段出来，然后放到时间线上
# 一般第三轨是音乐，上门两轨是放音效使用
# 放上音乐之后，需要结合音乐的感受，再对画面长短进行一些更细节的调整，这是很必要的
# 添加音效，如果您想让这两个片段之间的音效有过渡，就可以使用基本转场，让前面这个音效渐弱，同时我们还要控制音频素材的音量，这个音效我不想让它的声音太强，并且这个画面我既然做了复古，下面这里音频其实也要做复古一点，所以这个音效我先给他降低一点音量，控制到能听见一点点就好，OK接下来这个音乐因为是演示啊，我就随便先给它卡一下时长，到视频的结尾让它慢慢消失，然后就需要做声音效果了，我现在先把对应这个复古画面的音乐，给它剪出来，选中这个音乐片段，去右上角检查器，这个下面有一个效果叫做均衡器，我现在要对他进行消除高频的操作，在均衡器里只需要打开最右边的bond4，把这个点往左拉，右边这个蓝绿色的区域，就是我们去掉高频的部分

添加字幕

# 无论这个字体是你自己往上敲，还是别人给你一个logo图片，片子之中一般都会有一些文字出现，所以现在我们就去到左边标题，这个里面找到文本，然后给它拖上来，我就把这个一条放在开始，给它加上一个转场，这个字是缓缓出现的。
# 字幕摆放位置：一般都是放在视频轨道最上层的，剪辑软考里的字幕条，除了字以外，其他区域全都是透明的，这个透明的区域，我们一般把它叫作通道，接下来修改文字内容，我们现在选中这个字幕条，给它起一个名字叫做结晶展示，下面字体你也可以调，现在我换一个得意黑，然后包括字体大小字据位置，将字放在一个比较舒服的地方，这样的话添加字幕就完成了，做完片子之后一定要检查一遍，素材跟素材之间是没有夹帧还有黑帧的，画面跟声音全都确认没问题，这样的话我们就可以出片了。
50字体，间隔5，阴影x4,y-4,阴影4，75搞

导出

# 去到软件面板下放，有一个叫做交付的面板。我们首先在下方的时间线当中，就是要输出的段落，接下来我们就可以到上面，给它设置一个文件名，文件名还是按照年份日期加上项目名称，这个我就设置成230817结晶成篇，在位置点击浏览，选择一个存放目录，然后点击保存，
# 渲染：选择单个片段
# 导出视频：MP4格式，H264编解码器，1920x1080HD分辨率，25帧率，质量（成品20000，小样6000，【3840x2160的成片40000】）
# 音频和文件面板都是默认就好
# 然后就点击 “添加到渲染队列”，然后渲染队列里面手动选中作业，点开始渲染所有。片子就开始渲染输出了，渲染完的片子，我们就可以去到硬盘里的存放目录，就可以看到刚才我们输出的这个片子了。

一个片子从头到尾创作的大致流程了。

项目文件夹

## 整个项目的文件夹命令：年月日+项目名称，例如20260303麒麟操作系统XXX。如果需要另外一台电脑剪辑，直接将整个项目拷贝到另外一台电脑上就可以无缝衔接了。
所有的文件夹和文件的命名方式，都可以用年月日+关键词命名。防止覆盖
# 导出：里面分成了小样和成片。小样就是在剪辑过程中，导出的低质量成片，是给导演客户看的阶段成果，一般是微信传输；当剪辑全部完成，就导出定剪片或者成片。输出文件时可以使用v1或者v2区分版本
# 工程文件：存放达芬奇的项目或者是时间线，一般我们会在项目执行中后期导出项目或者是时间线，导出的时候也可以按照时间加项目名称来命名，防止来回修改。
# 脚本：如果视频有脚本，无论是分镜脚本还是文字脚本，或者参考片，都可以放在这里，
# 素材：这里面是你从网上下载的素材，并不是你原创的内容，里面包含图片视频音频包括背景音乐和音效。
# 特效 VFX：存放特效镜头
# 原始素材：这里面包含了原视频音频图片，这里存放实拍的任何素材，无论是用手机相机或者无人机的素材，命名我们会按照年月日+机型
# 字幕：一般国内的视频，都会把字幕做到片中，如果视频有双语，或者需要上传到国外的网站上，就需要单独导出外挂字幕文件，一般是.srt结尾
