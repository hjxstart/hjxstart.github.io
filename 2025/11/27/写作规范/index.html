通用规范

1 合规遵从原则

1.1 全球法律法规遵从

# 知识要求遵从国际标准和法律法规，包括但不限于如下要求。
如遵从广告法。需要特别注意的是知识中不得出现夸大、虚假宣传字眼，避免出现“最快”“最薄”“顶级”“全球销量第一”等极限词。

1.2 全球知识产权遵从

# 不得违反保密协议，不得透露内部信息。
# 未经授权，若使用第三方的知识时，不得使用他人保密信息；使用友商文档、其他出版物、或编排工具中的图标可能会涉及侵权。
# 在工作中不得使用无合法授权的第三方软件，包括但不限于商业软件、免费软件、开源软件等软件作品。比如：被A国制裁后，正式交付的知识不允许使用A国软件，知识中不允许有A国软件截图。
# 未经授权或未按照相关许可协议，不得使用第三方文字、音乐、视频、美术、摄影、字体、产品设计图等作品。
# 禁止任何可能泄露麒麟及客户、合作伙伴保密信息的行为。
# 不得诋毁友商产品，不得拔高或贬低产品功能。

1.3 全球文化遵从

# 避免使用成语、俗语、土话、行话、避免网络上转瞬即逝的时髦词汇。
反例：踩内存 duang 逼格 没有 剁手
正例：非法访问内存空间
# 避免使用令人不快的词语，避免涉及到人体缺陷。
反例：落后国家 农民工 打工仔 黑鬼 无聊的退休老人
正例：发展中国家 务工人员
# 避免使用笑话、幽默、有特定文化含义的比喻等修辞。 
# 避免使用宗教、政治导向词语，避免政治敏感人名、地点、网站、组织、国家，避免使用宗教节日。 
# 避免使用反映出性别、种族、肤色、宗教、政治的词汇或图案。
# 在通用版本中，应避免使用对某一国家或某种文化有特殊意义的样例、背景、图案、音视频。
# 含中文字符的Logo，在国际化和本地化时不作任何处理。
# 避免涉及个人具体信息、隐私等内容，以免让客户感觉冒犯。年龄、收入、婚姻、生活习惯、身体健康状况等属于个人隐私。
# 不得使用错误或有争议的国家名和地区名、地图。
