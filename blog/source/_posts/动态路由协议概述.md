## 根据作用范围可分: IGP/EGP

AS(自治系统): AS号(IANA组织管理，取值0-65535，私有AS号取值64512-65535).
IGP(内部网关协议): Interior Gateway Protocol, 在一个AS内部使用, 常用的协议有RIP、EIGRP、OSPF和IS-IS.
EGP(外部网关协议): Exterior Gateway Protocol, 多个AS之间使用, 常用的协议有BGP.

## 根据发现路由和计算路的方法可分: DV/LS

DV(距离矢量): Distance-Vector Protocol, 基于传闻的协议, 常用的协议有RIP、EIGRP和BGP.
LS(链路状态): Link-State Protocol, 基于拓扑的协议, 常用的协议有OSPF和IS-IS.

## DV协议细分Classfull与Classless

有类: 一条路由条目最多包含前缀和度量值(掩码靠猜, 下一条是IP数据包的源IP)。RIPv1
无类: 一条路由条目最多包含前缀、掩码、下一跳和度量值(出接口和管理距离是路由器不一样的)。RIPv2 EIGRP BGP

## OSPF 基本工作原理介绍

OSPF(开发式最短路径优先协议): Open Shortest Path First

# 静态路由高级特效

## 控制层面与数据层面负载均衡与冗余备份

```shell
1. 控制层面(哪条路由加表)

[冗余备份]浮动静态路由: 通过修改2条路由的管理距离来实现；
[负载均衡]等价负载均衡(ECMP): 两条2条一样的静态路由，管理距离没有修改(可以基于目的地，数据包来负载均衡)。

2. 数据层面(当路由加表后，用哪条路由转发数据)

[冗余备份] 主路经配置明细路由，备份路由配置汇总路由；
[负载均衡] 在冗余备份的情况下，对特定主机走备份链路再配置更详细的路由。
```

## 链路连通性检测工具SLA NQA与BFD

```shell
1. 原因: 静态路由消失的前提: 出接口down | 下一跳(没有去往吓一跳的直连路由)
检测2个节点之间的端到端连通性工具1: SLA服务级别协定
检测2个节点之间的端到端连通性工具2: NQA网络质量检测

2. SLA服务级别协定(工具，思科私有1)
在源/目的的端到端之间、设置Probe探针类型、发送探针的时间周期、连通性Timeout时间、连通性质量Threshold阈值。

3. NQA网络质量检测(工具，华为私有2)

4. BFD双向转发检测(公有协议)
可联动上层协议

```
