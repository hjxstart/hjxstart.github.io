### 20241104

1. 什么样是不好的问题

```bash
# 牛头不对马嘴的问题
# 可以通过简单的检索就能解决的问题
# 过度宽泛的问题
# 非开放性的问题
# 有引导性的问题
```

2. 什么样是好的问题

```bash
# 开放的，能够引起探讨和共同启发的问题；
# 一个好的问题是能让被提问者“一愣”的问题，一定是一个能够引起思考的开放性的没有引导性的问题；

## 面试例子
# 您希望找一个怎么样的人
# 您觉得我可以成为怎么样的人
```
