# 一、概述

```bash
园区网架构主要有：企业网园区、ISP园区和DC(数据中心)园区。

# 三级企业网架构
1. 接入层(Access)、汇聚层(Distribution)和核心层(Core)。
2. 接入层：包含楼层交换机(弱三层交换机，盒式交换机)，连接到客户端设备，具有价格低、数量多的特点
3. 汇聚层：提供网关和策略功能(三层交换机)，
4. 核心层：提供转发能力足够强(贵的交换机，框式交换机)，
```
