配置本机IP地址

# 右键网络图标，点击网络和Internet设置，在属性找到本地网络，配置IP地址

笔电连接到服务器Mgnt口

# 网线两端分别连接笔记本和RJ45接口和服务器后面的Mgnt管理接口

登录WEB界面

# 在浏览器上打开https:bmc_ip, 使用BMC的用户名和密码登录

配置RAID

# 点击系统管理下存储管理，配置磁盘raid，一般系统盘（2块一样大小固态盘，配置Raid1），数据盘（3块一样大小配置Raid5）
# 常用Raid1: 2快盘
# 常用Raid5: 最少3块盘
# 常用Raid6: 最少4块盘
