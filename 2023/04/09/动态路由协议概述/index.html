    1. жҺ§еҲ¶еұӮйқўдёҺж•°жҚ®еұӮйқўиҙҹиҪҪеқҮиЎЎдёҺеҶ—дҪҷеӨҮд»Ҫ

    4. и·Ҝз”ұйқһзӣҙиҝһдёӢдёҖи·ідёҺйҖ’еҪ’зҡ„еә”з”Ё

    • ж №жҚ®дҪңз”ЁиҢғеӣҙеҸҜеҲҶ: IGP/EGP

    AS(иҮӘжІ»зі»з»ҹ): ASеҸ·(IANAз»„з»Үз®ЎзҗҶпјҢеҸ–еҖј0-65535пјҢз§ҒжңүASеҸ·еҸ–еҖј64512-65535).
    IGP(еҶ…йғЁзҪ‘е…іеҚҸи®®): Interior Gateway Protocol, еңЁдёҖдёӘASеҶ…йғЁдҪҝз”Ё, еёёз”Ёзҡ„еҚҸи®®жңүRIPгҖҒEIGRPгҖҒOSPFе’ҢIS-IS.
    EGP(еӨ–йғЁзҪ‘е…іеҚҸи®®): Exterior Gateway Protocol, еӨҡдёӘASд№Ӣй—ҙдҪҝз”Ё, еёёз”Ёзҡ„еҚҸи®®жңүBGP.

    ж №жҚ®еҸ‘зҺ°и·Ҝз”ұе’Ңи®Ўз®—и·Ҝзҡ„ж–№жі•еҸҜеҲҶ: DV/LS

    DV(и·қзҰ»зҹўйҮҸ): Distance-Vector Protocol, еҹәдәҺдј й—»зҡ„еҚҸи®®, еёёз”Ёзҡ„еҚҸи®®жңүRIPгҖҒEIGRPе’ҢBGP.
    LS(й“ҫи·ҜзҠ¶жҖҒ): Link-State Protocol, еҹәдәҺжӢ“жү‘зҡ„еҚҸи®®, еёёз”Ёзҡ„еҚҸи®®жңүOSPFе’ҢIS-IS.

    DVеҚҸи®®з»ҶеҲҶClassfullдёҺClassless

    жңүзұ»: дёҖжқЎи·Ҝз”ұжқЎзӣ®жңҖеӨҡеҢ…еҗ«еүҚзјҖе’ҢеәҰйҮҸеҖј(жҺ©з Ғйқ зҢң, дёӢдёҖжқЎжҳҜIPж•°жҚ®еҢ…зҡ„жәҗIP)гҖӮRIPv1
    ж— зұ»: дёҖжқЎи·Ҝз”ұжқЎзӣ®жңҖеӨҡеҢ…еҗ«еүҚзјҖгҖҒжҺ©з ҒгҖҒдёӢдёҖи·іе’ҢеәҰйҮҸеҖј(еҮәжҺҘеҸЈе’Ңз®ЎзҗҶи·қзҰ»жҳҜи·Ҝз”ұеҷЁдёҚдёҖж ·зҡ„)гҖӮRIPv2 EIGRP BGP

    OSPF еҹәжң¬е·ҘдҪңеҺҹзҗҶд»Ӣз»Қ

    OSPF(ејҖеҸ‘ејҸжңҖзҹӯи·Ҝеҫ„дјҳе…ҲеҚҸи®®): Open Shortest Path First

2. ж•°жҚ®еұӮйқў(еҪ“и·Ҝз”ұеҠ иЎЁеҗҺпјҢз”Ёе“ӘжқЎи·Ҝз”ұиҪ¬еҸ‘ж•°жҚ®)
